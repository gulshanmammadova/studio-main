# **App Name**: CoffeeTeam Elite

## Core Features:

- Home Page Video Slider: Display a background video slider at the top of the home page.
- About Section with Image: Show a short description and an image about CoffeeTeam.
- Elite Gallery or Product Slider: Showcase a small gallery or a product slider featuring 3-4 products with a coffee-related quote.
- Categorized Menu Display: Display the menu items categorized into Tea, Coffee, and Desserts, with name and price.
- Menu Filtering and Sorting: Enable filtering of menu items by category and sorting by price.
- Product Showcase with Calculator: Display available coffee products with details, prices, images and add a simple 'calculator' function that sums selected products (no persistence).
- Direct Call/WhatsApp Reservation: Implement a reservation button that directly initiates a call or opens WhatsApp to a specified number.
- Location Map and Contact Information: Display the address on a map and the opening hours.

## Style Guidelines:

- Primary color: Dark, rich brown (#332200) for an elegant, coffee-inspired feel.
- Background color: Near-black (#121212) to complement the dark theme and give a premium look.
- Accent color: Soft gold (#D4A373) for highlights and call-to-action elements.
- Headline font: 'Mrs Saint Delafield', a handwritten serif, combined with Montserrat, Nunito, or sans-serif alternatives for responsive fallback.
- Body font: 'Montserrat', sans-serif for body text, ensuring readability.
- Use elegant, minimalist icons in gold or white to represent menu categories and features.
- Modern, responsive layout with a left-aligned filter section on desktop for the menu page.
- Subtle animations for transitions, button hover effects, and the gallery/slider to create a cool aesthetic.
